Descarga del Software MSI ERP

Paso 1: Acceder al sitio de descargas

Dirígete al portal oficial de Odoo para descargar la última versión del software.

Asegúrate de descargar la versión compatible con tu sistema operativo.

Paso 2: Seleccionar versión

Elige la versión adecuada según tus necesidades:

  • Versión Comunidad (gratuita)
  • Versión Enterprise (con soporte completo)

Paso 3: Completar formulario

Debes rellenar el formulario para iniciar la descarga

El tamaño del archivo es aproximadamente 2GB, dependiendo de la versión.

Paso 4: Descarga Iniciada

Aqui empezara a descargarse el software:

  1. Comprueba el tamaño del archivo descargado
  2. Verifica el checksum si está disponible

Aqui como se puede ver, se ha descargado el archivo de instalacion de Odoo:
