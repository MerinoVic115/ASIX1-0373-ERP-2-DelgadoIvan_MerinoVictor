Instalación del Software MSI ERP

Paso 1: Ejecutar el instalador

Localiza el archivo descargado y haz doble clic para iniciar el instalador.

Asegúrate de tener permisos de administrador.

Paso 2: Asistente de instalación

Sigue las instrucciones del asistente de instalación:

  1. Acepta los términos de licencia
  2. Selecciona la ubicación de instalación
  3. Elige los componentes a instalar

Paso 3: Instalar dependencias

El instalador detectará e instalará las dependencias necesarias:

  • PostgreSQL (para la base de datos)
  • Python (para el backend)
  • Otras librerías requeridas
No interrumpas este proceso.

Paso 4: Completar instalación

Una vez finalizada la instalación:

  1. Marca la opción para iniciar MSI ERP
  2. Haz clic en "Finalizar"
