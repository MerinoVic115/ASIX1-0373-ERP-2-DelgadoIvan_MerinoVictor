Acceso al Panel de Control de PostgreSQL

Guía para gestionar copias de seguridad y restauración de la base de datos MSI ERP.

Acceso a pgAdmin

  1. Busca "pgAdmin" en la barra de búsqueda de Windows
  2. Inicia la aplicación
  3. Inicia sesión con las credenciales configuradas durante la instalación de MSI ERP
Las credenciales por defecto suelen ser las mismas que para MSI ERP.

Crear copia de seguridad (Backup)

  1. Haz clic derecho sobre la base de datos que deseas respaldar
  2. Selecciona "Backup..."
  3. En la pestaña "General":
    • Asigna un nombre al backup
    • Selecciona el formato (por defecto "Custom")
  4. En la pestaña "Dump Options":
    • Marca "Blobs" si necesitas incluir archivos binarios
  5. Haz clic en los tres puntos para seleccionar ubicación del archivo
  6. Finalmente haz clic en "Execute"
No interrumpas el proceso hasta que finalice completamente.

Restaurar una base de datos

  1. Haz clic derecho sobre "Databases" o sobre una base de datos existente
  2. Selecciona "Restore..."
  3. En la pestaña "General":
    • Selecciona el archivo de backup (.backup o .sql)
    • Define el nombre de la nueva base de datos
  4. En la pestaña "Restore Options":
    • Marca "Clean before restore" para limpiar la base existente
  5. Haz clic en "Execute" para iniciar la restauración
El tiempo de restauración depende del tamaño de la base de datos.

Verificar la restauración

Una vez completado el proceso:

  1. Actualiza la lista de bases de datos
  2. Verifica que la nueva base aparece en la lista
  3. Conéctate a ella para comprobar que los datos están correctos
