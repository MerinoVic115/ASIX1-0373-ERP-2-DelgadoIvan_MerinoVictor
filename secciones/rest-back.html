Copia de Seguridad y Restauración de Base de Datos

Acceso a pgAdmin

Para gestionar las copias de seguridad necesitamos acceder a pgAdmin, la herramienta de administración de PostgreSQL:

  1. Buscar "pgAdmin" en la barra de búsqueda de Windows
  2. Iniciar sesión con las credenciales configuradas durante la instalación de Odoo

Crear una copia de seguridad (Backup)

  1. Haz clic derecho sobre la base de datos que quieres respaldar
  2. Selecciona la opción "Backup"
  3. En el primer recuadro, haz clic en los 3 puntos para seleccionar la ubicación del archivo
  4. Haz clic en "Execute" para iniciar el proceso

Restaurar una base de datos

El proceso de restauración es similar al de backup:

  1. Haz clic derecho sobre la base de datos o sobre "Databases"
  2. Selecciona la opción "Restore"
  3. Selecciona el archivo de backup que quieres restaurar
  4. Haz clic en "Execute" para iniciar la restauración
